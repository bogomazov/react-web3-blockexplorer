# react-web3-blockexplorer

## To start

1.  Git clone

```javascript
git clone https://github.com/Glebasya2000/react-web3-blockexplorer.git
```

2.  Yarn initialization in proect folder (Yarn has to be already installed on computer)

```javascript
yarn;
```

3.  Start project

```javascript
yarn dev
```

4.  Go to localhost:8080
